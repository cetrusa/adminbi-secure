Cubos de Ventas
Análisis comercial, reportes y trazabilidad
{% if perms.permisos.cubo %} {% endif %} {% if perms.permisos.proveedor %} {% endif %} {% if perms.permisos.faltantes %} {% endif %} {% if perms.permisos.preventa %} {% endif %} {% if perms.permisos.trazabilidad %} {% endif %} {% if perms.permisos.amovildesk %} {% endif %} {% if perms.permisos.reportes %} {% endif %} {% if perms.permisos.matrix %} {% endif %} {% if perms.permisos.config_email_reportes %} {% endif %}
Informe Descripción
Cubo de Ventas Genera el cubo de ventas consolidado a partir de la base de datos SIDIS. Permite seleccionar rango de fechas, empresa y base de datos para generar archivos descargables con el análisis de ventas por producto, cliente y territorio.
Genera archivo Excel descargable
Inf. Proveedor Genera informes de ventas segmentados por proveedor. Útil para compartir información de desempeño comercial con proveedores específicos, incluyendo volúmenes, tendencias y distribución por zona.
Genera archivo Excel descargable
Faltantes Identifica productos con quiebre de stock o faltantes en el inventario. Muestra productos que no se han vendido o que tienen disponibilidad cero, comparando contra el catálogo activo. Permite filtrar por proveedor, categoría y período.
Preventa Reporte de pedidos preventa capturados por la fuerza de ventas. Permite analizar la conversión de preventa a facturación y el cumplimiento de metas por vendedor, zona y período.
Trazabilidad Compara pedidos de preventa contra facturación real. Muestra KPIs de efectividad: porcentaje de conversión, diferencias entre lo pedido y lo facturado, motivos de rechazo y cumplimiento por vendedor. Incluye vista detallada con drill-down.
Inf. Amovildesk Dashboard integrado con información de la plataforma Amovildesk para seguimiento de gestión comercial y logística de campo. Visualiza métricas de operación móvil.
Reporteador Generador de reportes personalizados. Selecciona una plantilla de reporte predefinida, aplica filtros de fecha y base de datos, y descarga los resultados en formato Excel. Las plantillas disponibles se configuran desde el administrador.
Genera archivo Excel descargable
Matrix Ventas Genera una matriz cruzada de ventas (productos vs clientes/territorios). Ideal para análisis de cobertura y penetración de mercado por categoría de producto y zona geográfica.
Genera archivo Excel descargable
Config. Correos Administra las listas de distribución de correo electrónico. Gestiona contactos de proveedores y supervisores para el envío programado de reportes. Permite agregar individualmente o carga masiva desde Excel.

Actualización
Procesos de sincronización de datos y cargues manuales
{% if perms.permisos.actualizar_base %} {% endif %} {% if perms.permisos.actualizacion_bi %} {% endif %} {% if perms.permisos.cargue_infoventas %} {% endif %} {% if perms.permisos.cargue_maestras or perms.permisos.cargue_infoproducto %} {% endif %}
Proceso Descripción
Actualización BD Ejecuta el proceso ETL de extracción e inserción de datos desde la base origen (SIDIS) hacia la base BI. Actualiza tablas de ventas, clientes, productos, bodegas y catálogos para el rango de fechas seleccionado. El proceso corre en segundo plano y notifica al completar.
Proceso en segundo plano - puede tardar varios minutos
Actualización BI Refresca el modelo de datos BI después de una actualización de base de datos. Recalcula indicadores, agrega nuevos períodos y sincroniza los dashboards de Power BI con los datos más recientes.
Cargue Info Ventas Permite cargar manualmente archivos de información de ventas cuando la extracción automática no está disponible. Acepta archivos Excel con el formato estándar de ventas definido por el sistema.
Formato: Excel (.xlsx)
Archivos Maestros Carga y actualiza las tablas maestras del sistema: productos, proveedores, categorías y otros catálogos base. Permite importar desde archivos Excel para mantener los datos de referencia actualizados.
Formato: Excel (.xlsx)

Bimbo
Reportes y herramientas para la operación Bimbo
Informe Descripción
Venta Cero Analiza productos Bimbo con venta cero en el período seleccionado. Identifica SKUs sin movimiento por agencia (CEVE), permitiendo detectar problemas de distribución, disponibilidad o discontinuación de productos.
Rutero Reporte de cobertura de rutas por territorio y agencia. Muestra la asignación de clientes a rutas de visita y el nivel de cobertura de la fuerza de ventas Bimbo en cada zona.
Inventarios Control de niveles de inventario de productos Bimbo por agencia. Muestra existencias actuales, indica rotación y genera alertas de sobre-stock o desabastecimiento.
Preventa Seguimiento de pedidos preventa para agencias Bimbo. Compara lo solicitado contra lo despachado y calcula indicadores de cumplimiento y efectividad por ruta.
Faltantes Detalla productos Bimbo faltantes o con quiebre de inventario por agencia. Prioriza productos para reabastecimiento y mide niveles de servicio por CEVE.
Planos Bimbo Genera archivos planos específicos para integración con los sistemas Bimbo. Incluye formatos de ventas, inventarios y catálogos en las estructuras requeridas por la operación.
Genera archivos descargables
Equivalencias Gestiona la tabla de equivalencias entre códigos de producto internos y códigos Bimbo. Permite crear y editar mapeos de productos manualmente para asegurar la correcta correspondencia en reportes cruzados.
Homologación Proceso automático y manual de homologación de productos. El sistema sugiere equivalencias por similitud de nombre y código (Discovery + Snapshot), y permite ajustar manualmente los mapeos cuando la coincidencia automática no es exacta.
